# E-Commerce Customer Analytics Platform

An end-to-end data science project analysing real e-commerce 
data with customer segmentation, churn prediction and 
sentiment analysis.

## Tech Stack
- Python, Pandas, NumPy
- Scikit-learn, XGBoost (Machine Learning)
- TextBlob (NLP & Sentiment Analysis)
- Plotly (Interactive Visualisation)
- Streamlit (Dashboard)

## Features
- Business Overview — revenue trends, top products, top countries
- Customer Segmentation — KMeans clustering with RFM analysis
- Churn Prediction — XGBoost model with probability scores
- Sentiment Analysis — product sentiment + live review analyser

## How to Run
1. Clone this repo
2. Install dependencies: pip install -r requirements.txt
3. Add Online Retail.xlsx to the project folder
4. Run: streamlit run ecommerce_dashboard.py

## Dataset
UCI Online Retail Dataset — 500,000+ real UK e-commerce 
transactions. Available at:
https://archive.ics.uci.edu/dataset/352/online+retail

## Background
Built as a portfolio project following MSc Business Analytics
(University of Greenwich), demonstrating end-to-end data 
science skills across ML, NLP, and dashboard deployment.
